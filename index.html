What is the SHARE IT Act?

The Source Code Harmonization And Reuse in Information Technology Act (SHARE IT Act) requires federal agencies to publicly share repositories of custom-developed source code to eliminate duplicative work and promote re-use.



This website provides the following to help users become SHARE IT Act ready.
